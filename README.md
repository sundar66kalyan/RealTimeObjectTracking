\# Real-Time Intelligent Object Tracking and Video Analytics System



\## Features



\- YOLOv8 Object Detection

\- ByteTrack Multi-Object Tracking

\- Vehicle Counting

\- Line Crossing Counter

\- Speed Estimation

\- Vehicle Analytics

\- CSV Export

\- Analytics Report Generation

\- Historical Analytics

\- Alert System

\- Video Export

\- Streamlit Dashboard



\## Installation



pip install -r requirements.txt



\## Run



python src/download\_model.py



python src/track.py



python src/vehicle\_count.py



python src/speed\_estimation.py



streamlit run dashboard/app.py



\## Technology Stack



\- Python

\- YOLOv8

\- ByteTrack

\- OpenCV

\- Streamlit

\- Pandas
